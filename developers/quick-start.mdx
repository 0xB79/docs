First, go to Livepeer Studio, if you haven't already, and create an account. Once you've created an account, you'll be able to create an API key by clicking on the "Create API Key" button on Developers page.

<Frame>
  ![Livepeer Studio - Create API key
  page](/images/tutorials/studio-create-api.png)
</Frame>

You can now use this API keys in Livepeer SDKs and APIs in order to authenticate your requests and start building.

## Install the SDK

<Tabs>
  <Tab title="Node.js">
  ```javascript 
  npm install livepeer 
  ```
  </Tab>
  <Tab title="Python">
  ```sh 
  pip install livepeer 
  ```
  </Tab>
  <Tab title="Ruby">
  ```sh 
  gem install livepeer 
  ```
  </Tab>
  <Tab title="PHP">
  ```sh 
  composer require livepeer/livepeer-php 
  ```
  </Tab>
  <Tab title="Go">
  ```sh 
  go get github.com/livepeer/livepeer-go-SDK 
  ```
  </Tab>
</Tabs>

## Configure the SDK

<Tabs>
  <Tab title="Node.js">
    ```javascript
    import { Livepeer } from "livepeer";

    const apiKey = 'YOUR_API_KEY'; // Replace with your API key

    const livepeer = new Livepeer(apiKey);
    ```

  </Tab>

  <Tab title="Python">
    ```python
    from livepeer import Livepeer

    # Initialize the Livepeer client with your API key
    api_key = "YOUR_API_KEY"
    livepeer = Livepeer(api_key)
    ```

  </Tab>

  <Tab title="Ruby">
    ```ruby
    require 'livepeer'

    # Initialize the Livepeer client with your API key
    api_key = 'YOUR_API_KEY'
    client = Livepeer::Client.new(api_key: api_key)
    ```

  </Tab>

  <Tab title="PHP">
    ```php
    <?php

    require 'vendor/autoload.php';

    use Livepeer\Livepeer;

    // Initialize the Livepeer client with your API key
    $api_key = 'YOUR_API_KEY';
    $livepeer = new Livepeer($api_key);
    ```

  </Tab>

  <Tab title="Go">
    ```go
    package main

    import (
      "github.com/livepeer/go-livepeer/livepeer"
    )

    func main() {
      // Initialize the Livepeer client with your API key
      apiKey := "YOUR_API_KEY"
      client := livepeer.NewLivepeerClient(apiKey)
    }
    ```

  </Tab>
</Tabs>

## Start building

That is it, you are ready to start building! Check out the [SDKs](/sdk) and [APIs](/api) pages to learn more about how to use the SDKs and APIs to build your application.

You can also refer to the [Guides](/guides) section for more in-depth tutorials on how to use the SDKs and APIs to build specific applications.
