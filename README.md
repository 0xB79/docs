# Livepeer Documentation

These docs are visible at (readthedocs.io).

## Building Locally

These docs are generated using Sphinx.

```
$ pip install sphinx
$ git clone https://github.com/livepeer/docs.git
$ cd docs/docs
$ make html
$ open build/html/index.html
```
